我的第一个博客文章

1.GitHub创建 https://blog.csdn.net/qq_26927285/article/details/78762237
2.Hexo使用
https://www.bilibili.com/video/av44544186?share_medium=android&share_source=copy_link&bbid=OAg7Wm9eblZuW2oOcg5yinfoc&ts=1553139381076

2.1 先下载安装node node -v 查看版本 npm -v 查看版本
2.2链接淘宝镜像安装cnpm npm install -g cnpm –registry =https://registry.npm.taobao.org
2.3安装 cnpm install -g hexo-cli 查看版本 hexo -v （下载安装 git）
2.4创建文件夹blog 在文件夹下执行命令 >hexo init （自动下载安装git）
2.5启动博客 hexo s 本地4000端口
你的电脑端口被占用了。
hexo默认的端口是4000，如果你的电脑安装了福昕阅读器，，就是他，没错，坑爹吧！！！！
启动hexo s 的时候，用这个命令，换一个端口。
hexo s -p 5000
换成5000端口，果断可以访问。
2.6创建博客 >hexo n “我的第一篇博客文章”
2.7 找到该文件去修改
2.8 回到 blog 目录 hexo clean 清理
2.9Hexo g 自动生成网站静态文件
2.10Hexo s -p 5001 启动 可以在本地浏览了
2.11链接git 创建git 库
2.12创建好后在 hexo 的blog 目录下装git部署插件 命令为cnpm install –save hexo-deployer-git
2.13设置 _config.yml 进入
